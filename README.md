Overview

This project demonstrates an end-to-end data analytics workflow, covering data extraction, exploratory analysis, cleaning, SQL querying, visualization, and business reporting.
The goal is to analyze customer shopping behavior and generate actionable insights using Python, SQL, and Power BI.

The project is designed to reflect real-world analytics work and is suitable for showcasing to recruiters and hiring managers.

Dataset

Source: Customer Shopping Behavior dataset (CSV)

Content Includes:

Customer demographics

Purchase frequency and spending patterns

Product categories

Transaction-level information

The dataset is loaded and analyzed using Python, and later queried using SQL after being stored in a relational database.

Tools & Technologies

Python: Data loading, EDA, and data cleaning

Libraries: Pandas, NumPy, Matplotlib, Seaborn

SQL: PostgreSQL / MySQL / SQL Server

Used for analytical queries and aggregations

Power BI: Interactive dashboard creation

Gamma: Presentation (PPT) creation

Jupyter Notebook: Analysis and documentation

Project Workflow / Steps

Data Loading

Imported raw CSV data into Python using Pandas

Exploratory Data Analysis (EDA)

Checked data structure, missing values, and distributions

Identified trends in customer behavior and spending

Data Cleaning

Handled missing and inconsistent values

Standardized columns for analysis and SQL usage

SQL Analysis

Loaded cleaned data into a relational database

Ran analytical SQL queries to answer business questions

Performed aggregations, filtering, and grouping

Dashboard Development

Built an interactive Power BI dashboard

Visualized KPIs, trends, and customer segments

Reporting & Presentation

Created a business report summarizing insights

Designed a PPT using Gamma for stakeholder communication

Dashboard

The Power BI dashboard provides:

Customer segmentation insights

Purchase trends over time

Category-wise and spending analysis

Key performance indicators (KPIs)

The dashboard is designed to be interactive and business-focused, enabling quick decision-making.

Key Results & Insights
